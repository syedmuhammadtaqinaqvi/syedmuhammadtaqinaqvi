# âš¡ GitHub Action to Auto-Update "Currently"

Create file: `.github/workflows/update-readme.yml`

```yaml
name: Update README - Currently
on:
  push:
    paths:
      - '.github/README_DATA/currently.md'
  schedule:
    - cron: '0 2 * * *'

permissions:
  contents: write

jobs:
  update-readme:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - run: |
          git config --global user.name "github-actions[bot]"
          git config --global user.email "github-actions[bot]@users.noreply.github.com"

          CONTENT=$(cat .github/README_DATA/currently.md || echo "Learning Flutter & AI â€¢ Building apps")
          sed -i "/<!--CURRENTLY-->/,/\*\*Currently:/c\<!--CURRENTLY-->\n**Currently:** $CONTENT\n" README.md

          git add README.md
          git commit -m "chore: update Currently section" || echo "no changes"
          git push
